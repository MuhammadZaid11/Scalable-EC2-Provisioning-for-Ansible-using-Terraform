# 🚀 Infrastructure for Ansible using Terraform

This project demonstrates how to build scalable infrastructure for Ansible using Terraform on AWS.

---

## 📌 Project Overview

The goal of this project is to automate the provisioning of EC2 instances that can be used as Ansible managed nodes.

---

## 🏗️ Architecture

Terraform provisions multiple EC2 instances dynamically using reusable modules.

- AWS EC2 instances
- Terraform modules
- SSH key-based authentication
- Scalable infrastructure using variables

---

## ⚙️ Technologies Used

- Terraform
- AWS (EC2)
- Ansible (Planned Integration)
- Linux (Ubuntu)
